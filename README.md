# 🏅 SkillSphere – Learning Resource Website

## 📌 Project Overview

SkillSphere is a web-based application that helps users discover useful coding resources such as courses, tutorials, and documentation. It provides a clean and interactive interface where users can search, filter, and sort resources efficiently.

The website fetches real-time data from a public API and displays it dynamically using JavaScript.

---

## 🎯 Objective

The objective of this project is to demonstrate:

* API integration using Fetch API
* Use of JavaScript array higher-order functions
* Dynamic UI rendering
* Interactive and responsive website design

---

## 🌐 API Used

* Coding Resources API  
  https://api.sampleapis.com/codingresources/codingResources

---

## 🚀 Features

### 🔍 Search Functionality

* Users can search resources by title or description

### 🎯 Filtering

* Filter resources based on website/source

### 🔃 Sorting

* Sort resources alphabetically:

  * A → Z
  * Z → A

### ⭐ Favorites

* Users can mark resources as favorites  
* Favorites are stored using localStorage  

### 📄 Pagination

* Resources are divided into multiple pages for better viewing  

### 📚 Resource Display

Each resource includes:

* Title  
* Source (website name)  

### 🔗 Resource Navigation

* Clicking on "Open Resource" opens the learning website in a new tab  

### 👁 View More

* Displays full resource description using alert  

### 🌙 Dark Mode

* Toggle between light and dark themes  

### 📱 Responsive Design

* Fully responsive across mobile, tablet, and desktop
